# 虚拟环境与包管理

## 一、虚拟环境

## 1、**Virtualenv**

### pip安装

```python
 pip install virtualenv
```

### 创建虚拟环境

```python
virtualenv venv
```

这会创建`ENV/lib/`、`ENV/include`、`ENV/bin`（Windows系统下是`ENV/Scripts`）几个文件夹，它们共同构成了一个完整的Python环境。在`ENV/lib/`中有pip等工具用于安装第三方包，它们和系统中已安装的Python互不关联。

这里有一个额外参数`--system-site-packages`，如果在创建虚拟环境的时候使用该参数，虚拟环境会继承主环境的所有第三方包。如果你希望虚拟环境和主环境独立，就不要用这个参数。

要激活创建的虚拟环境，需要执行虚拟环境其中的activate脚本（<font style="color:#F5222D;">注意Windows系统和Linux系统间的差别</font>）：

```python
\path\to\env\Scripts\activate  # cmd
```

<font style="color:#404040;">这样一来就进入到虚拟环境中去了，此时所有的Python命令和安装的pip包，全部都在虚拟环境中，不会影响主Python环境。</font>

<font style="color:#404040;"></font>

## 二、pip的使用

### 1、更换pip到国内镜像

* <font style="color:rgb(0, 0, 0);">清华：https://pypi.tuna.tsinghua.edu.cn/simple</font>
* <font style="color:rgb(0, 0, 0);">阿里云：http://mirrors.aliyun.com/pypi/simple/</font>
* <font style="color:rgb(0, 0, 0);">中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/</font>
* <font style="color:rgb(0, 0, 0);">华中理工大学：http://pypi.hustunique.com/</font>
* <font style="color:rgb(0, 0, 0);">山东理工大学：http://pypi.sdutlinux.org/</font>
* <font style="color:rgb(0, 0, 0);">豆瓣：http://pypi.douban.com/simple/</font>

<font style="color:rgb(0, 0, 0);">如果是临时性的，可以使用pip参数 -i + 国内镜像源地址 + 库名，进行安装。</font>

```shell
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
```

<font style="color:rgb(0, 0, 0);">如果要永久性的，可以按以下操作。</font>

<font style="color:rgb(0, 0, 0);">Linux下，修改 </font><code><font style="color:rgb(0, 0, 0);">~/.pip/pip.conf</font></code><font style="color:rgb(0, 0, 0);"> (没有就创建一个，文件夹要加“.”，表示是隐藏文件夹)。</font>

<font style="color:rgb(0, 0, 0);">Windows下，直接在user目录中创建一个pip目录，如：</font><code><font style="color:rgb(0, 0, 0);">C:\Users\xx\pip</font></code><font style="color:rgb(0, 0, 0);">，</font><code><font style="color:rgb(0, 0, 0);">xx</font></code><font style="color:rgb(0, 0, 0);">一般表示用户名，新建文件</font><code><font style="color:rgb(0, 0, 0);">pip.ini</font></code><font style="color:rgb(0, 0, 0);">。</font>

<font style="color:rgb(0, 0, 0);">它们的文件内容如下（使用的是清华镜像，其他可以自行修改）：</font>

```java
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
[install]
trusted-host=mirrors.aliyun.com
```

### 2、显示版本和路径

```shell
pip --version
```

### 3、升级pip

```shell
pip install -U pip
```

### 4、安装包

```shell
pip install SomePackage              # 最新版本
pip install SomePackage==1.0.4       # 指定版本
pip install 'SomePackage>=1.0.4'     # 最小版本
```

### 5、升级包

```shell
pip install --upgrade SomePackage
```

### 6、卸载包

```shell
pip uninstall SomePackage
```

### 7、搜索包

```shell
pip search SomePackage
```

### 8、列出已安装的包

```shell
pip list
```

### 9、查看可升级的包

```shell
pip list -o
```
