[阿德/nvm](https://gitee.com/sdq/nvm)

```bash
nvm ls

# 列出所有可安装版本
nvm ls-remote

# 安装某个版本Node
nvm install lts/fermium
nvm install v12.20.1
nvm install v15.5.1

# 切换Node版本
nvm use system
nvm use 14.15    # 不用全部打出版本号

# 更新nvm
nvm-update


# 查看配置
npm config ls

npm config set registry https://registry.npm.taobao.org 
```
